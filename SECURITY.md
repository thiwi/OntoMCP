# Security Policy

## Supported Versions

At the moment, only the latest `main` branch and the most recent GitHub release
are considered supported for security fixes.

## Reporting a Vulnerability

Please do **not** open public issues for vulnerabilities.

Report security issues to:

- Email: **security@thiwi.dev**
- Subject: `OntoMCP Security Report`

Please include:

- affected version / commit SHA
- reproduction steps or proof-of-concept
- expected impact and affected component

We will acknowledge receipt within 72 hours and provide status updates during
triage and remediation.
