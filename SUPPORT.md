# Support

## Questions and Usage Help

Use GitHub Discussions or Issues:

- Questions: https://github.com/ontomcp/ontomcp/discussions
- Bugs/Features: https://github.com/ontomcp/ontomcp/issues

## Bug Reports

Please include:

- ontology snippet (`.ttl`)
- tool invocation payload
- expected behavior and actual behavior
- Node.js / npm version and commit SHA

## Security Issues

Do not report security issues in public trackers.
See [SECURITY.md](./SECURITY.md).
